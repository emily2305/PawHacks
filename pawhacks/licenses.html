Licenses

Dental Hygienist -
Associate Degree from CODA Accredited Dental Hygienist Program & Pass Written and Clinical Exams
All States Requires License - For detail check State Dental Board

Dentist -
Doctorate from CODA Accredited Dental Program & Pass Written and Clinical Exams
All States Requires License - For detail check State Dental Board

Dietitian or Nutritionist -
Bachelor's Degree from ACEND Accredited Nutrition Program & Complete Supervised Practice and Pass Exam
Most States Requires License - For detail check State Nutrition Board

Doctor or Surgeon -
Doctorate from LCME Accredited Medical School & Complete Residency and Pass Exam
All States Requires License - For detail check State Medical Board

Licensed Practical Nurse (LPN) / Licensed Vocational Nurse (LVN) -
Certificate or Diploma from ACEN Accredited Practical Nursing School & Pass NCLEXPN Exam
All States Requires License - For detail check State Nursing Board

Registered Nurse (RN) -
Associate or Bachelor's Degree from ACEN and CCNE Accredited RN program & Pass NCLEXPN Exam
All States Requires License - For detail check State Nursing Board

Pharmacist -
Doctorate from ACPE Accredited Pharmacy program & Pass 2 Exams
All States Requires License - For detail check State Pharmacy Board

CPA -
150 postsecondary credits and pass the CPA exam
For detail check National Association of State Boards of Accountancy

K-12 Teachers -
Bachelor's Degree from Accredited Teaching Program then State-issued License, must pass exams and apply for teaching license or certification from State
For detail check State Department of Education for Licensing Requirements

Principals & School Administrators -
Principals need a Master's in Education
For detail check State Department of Education for Licensing Requirements

Clinical Social Worker -
Master's Degree from an CSWE Accredited Social Work Program, Complete Supervised Clinical Experience, and Pass a Clinical Exam
All States Requires License - For detail check State Social Work Board

Marriage and Family Therapist -
Master's Degree from an CACREP, AAMFT, MPCAC Accredited Therapy Program, Complete Supervised Clinical Experience, and Pass an Exam
All States Requires License - For detail check State Therapy Board
